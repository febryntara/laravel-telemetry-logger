# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-01

### Added
- Initial release
- HTTP request/response logging (all methods)
- Automatic sensitive field and header redaction
- Exception logging with stack traces
- Slow query detection
- Queue job logging
- Artisan command logging
- Async dispatch via Laravel Queue
- Retry with backoff on microservice failure
- Custom user resolver interface
- Session data capture
- Route exclusion patterns
